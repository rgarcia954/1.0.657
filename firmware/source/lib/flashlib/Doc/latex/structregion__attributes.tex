\doxysubsection{Detailed Description}
Flash region attributes. 

Each flash bank consists of Code, Data and N\+VR regions~\newline
 Code region has 512 x 38 Bits/\+Sector, and total size = 88K x 38 Bits~\newline
 Data region is 64 x 38 Bits/\+Sector, and total size = 40K x 38 Bits~\newline
 N\+VR region is 64 x 38 Bits/\+Sector, and total size = 64K x 38 Bits~\newline
 \begin{DoxyNote}{Note}
N\+VR\mbox{[}0\+:3\mbox{]} is reserved for Cryptocell and cannot be accessed~\newline

\end{DoxyNote}
Flash programming is done in rows. Each programmable sector on flash can consist of one or multiple rows.

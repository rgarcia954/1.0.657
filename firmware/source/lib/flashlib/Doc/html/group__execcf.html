__STATIC_INLINE void Sys_Flash_EndOperationWaitIdle ( FLASH_Type *  flash)

End ongoing flash command operation if interface is busy or sequential write operation is active
and wait until flash interface is free again.

Note
CMD_END command is sent to the interface if found busy to end any ongoing operation

__STATIC_INLINE void Sys_Flash_ExecuteCommand ( FLASH_Type *  flash,
uint32_t  cmd 
)

Execute a command on the flash interface This function checks if the interface is free before initiating a flash command by executing Sys_Flash_ExecutePrecondFlashCommand
After a command is sent to the interface the function polls and waits for the command to be completed using Sys_Flash_IF_WaitBusy.

Parameters
[in]flashflash instance being used
[in]cmdcommand to execute
Note
application should be careful in changing bits on IF_CTRL register to avoid unnecessary change of flash state
Changing retry bits does not initiate a command on the flash interface hence this command does not poll the busy bit

Execute a sequential command on the flash interface This function does not check if the interface is free nor does it poll for busy bit in the flash interface.

Parameters
[in]flashflash instance being used
[in]cmdcommand to execute

Warning: Do not wait for busy when issuing a command for sequential operation as busy will not be cleared until the sequential operation is completed by END command

FlashStatus_t Flash_ReadDouble ( uint32_t  addr,
uint32_t *  word 
)

Read a 38-bit word from flash.

This function temporarily disables automatic flash ECC generation in hardware,
thus allowing the user to read 38-bits from a single word address in flash.

Note
ECC check is not performed in this function on the 32-bits. Application should use other mechanism to verify ECC if necessary
Parameters
[in]addraddress to read the 38-bit word
[out]word32-bit C pointer to store the word which is read from flash,
word[0] [0:31] bits consist of first lower 32-bits
word[1] [0:6] bits consist of the remaining higher 6-bits
Returns
Flash API status code
See also
FlashStatus_t
Note
addr should be word aligned
CPU interrupts are disabled momentarily through PRIMASK register

FlashStatus_t Flash_WriteBuffer ( uint32_t  addr,
uint32_t  word_length,
uint32_t *  words,
bool  enb_endurance 
)

Write contents of a static memory buffer to flash.

This function writes the contents of static memory buffer to flash. A verification of write is performed after write to ensure write was successful

Parameters
[in]addraddress of first word location in flash
[in]word_lengthtotal number of words to be written to flash
[in]wordsa 32-bit C pointer to the memory location of the buffer to
be written to flash
[in]enb_enduranceset to 0 for default flash endurance
set to 1 to enable two stage programming for higher endurance of data programmed
Returns
Flash API status code
See also
FlashStatus_t
Note
addr should be word aligned
Contents of flash must be erased prior to performing a write
CPU interrupts are disabled momentarily through PRIMASK register to ensure that the flash
is not exposed to higher programming voltages
When enb_endurance is enabled two stage programming is done.
Application must ensure that the function completes and that the
return value is FLASH_ERR_NONE to consider the two stage programming
to be done.

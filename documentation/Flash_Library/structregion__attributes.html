Detailed Description

Flash region attributes.

Each flash bank consists of Code, Data and NVR regions
Code region has 512 x 38 Bits/Sector, and total size = 88K x 38 Bits
Data region is 64 x 38 Bits/Sector, and total size = 40K x 38 Bits
NVR region is 64 x 38 Bits/Sector, and total size = 64K x 38 Bits

Note
NVR[0:3] is reserved for Cryptocell and cannot be accessed

Flash programming is done in rows. Each programmable sector on flash can consist of one or multiple rows.

Note: On Flash0 NVR[3:0]
